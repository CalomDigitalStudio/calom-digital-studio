# Calom Digital Studio — Premium Website

Sito statico premium pronto per deploy.

## Come vederlo
1. Estrai lo ZIP.
2. Apri `index.html` con Chrome o Safari.

## Come pubblicarlo
Carica tutti i file su Vercel, Netlify, Cloudflare Pages o hosting tradizionale.

## File principali
- `index.html`
- `style.css`
- `script.js`
- `pages/`
- `assets/calom-logo.svg`

## Aggiornamento commerciale
Sono stati integrati i percorsi START, GROWTH e PRO senza prezzi:
- START: per iniziare
- GROWTH: per vendere bene
- PRO: per crescere e scalare

La logica comunica differenza di valore tramite personalizzazione, strategia e maturità del progetto.
